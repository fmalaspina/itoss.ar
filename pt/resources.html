Docker instructions​

The easiest way to get itoss up and running is using our docker compose image.

mkdir /tmp/itoss
cd /tmp/itoss
wget https://github.com/fmalaspina/itoss.ar/releases/download/v5.2/docker-compose.yml
docker compose up
